# ASS-LIGHT Android

Native Android implementation of ASS-LIGHT: randomized original sound events, variable volume, five chaos profiles, persistent background operation, direct flashlight control, optional event pulses, and a one-tap Portal Seal.

The GitHub workflow compiles a signed APK for direct installation and an AAB candidate for later Google Play submission. The direct-install RC signature is intentionally non-production; the future confidential Play upload key remains a separate blocked gate.
