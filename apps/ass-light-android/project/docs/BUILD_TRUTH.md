# ASS-LIGHT Android RC1 Build Truth

## Active baseline

Native Android Java application, package `com.ghostatlas.asslight.rc`, compile and target SDK 35, minimum SDK 23, Java 17.

## Automated proof

GitHub Actions regenerates all twelve original WAV assets, runs source-definition and denylist audits, compiles and lints the app, verifies APK signing and package declarations, installs and launches on an Android 15 emulator, runs MediaPlayer and foreground-service instrumentation tests, and emits staged SECA receipts.

## Explicit boundaries

A green automated run authorizes only a direct-install Android release candidate. It does not prove physical speaker quality, rear flashlight output, lock-screen scheduling, manufacturer battery behavior, private Google Play signing, or Play Console approval.

## Promotion law

No Store-ready, production, complete, or physically verified claim may be made until the corresponding signed or human-in-the-loop receipts exist.
