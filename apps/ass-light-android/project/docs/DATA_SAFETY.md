# Google Play Data Safety Draft

- Data collected: None.
- Data shared: None.
- Data encrypted in transit: Not applicable because the app has no network channel.
- Account deletion: Not applicable because the app has no accounts.
- Camera permission purpose: rear flashlight control only; no image or video capture.
- Foreground service purpose: user-started randomized media playback with a visible ongoing notification and immediate stop action.
- Advertising: None.
- Analytics: None.
