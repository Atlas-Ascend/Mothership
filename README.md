# Mothership

Ghost Atlas public application delivery repository.

## ASS-LIGHT Android RC1

ASS-LIGHT is a user-started Android novelty app that plays original randomized comedy sounds at randomized intervals and volume levels, with optional flashlight pulses.

This repository publishes the direct-install Android release candidate, source packet, SECA audit receipts, checksums, and manual device validation form.

### Build truth

The release workflow must pass source, artifact, emulator-runtime, signing, installation, and launch gates before it publishes an APK. Physical speaker quality, flashlight behavior, lock-screen continuity, and OEM battery behavior remain manual phone proof gates and are never represented as automatically verified.
